## Short Description
JobSwipe is a web application that helps users find jobs using a swipe interaction similar to Tinder. Swipe right to save a job, swipe left to skip it.

## Key Features

| Feature | File | Description |
|---------|------|-------------|
| Job Swiping | `src/components/JobSwipe.jsx` | Swipe left/right to filter jobs, view details, filter by industry/location |
| Saved Jobs | `src/App.jsx` | List of saved jobs, with the ability to remove them |
| Feed | `src/components/FeedPage.jsx` | Update feed |
| Learning | `src/components/LearningHubPage.jsx` | Learning center |
| Advisor | `src/components/AdvisorProfilePage.jsx` | Advice from well-known mentors |
| Profile | `src/components/ProfilePage.jsx` | Personal profile page |
| CV Editing | `src/components/CVFixerPage.jsx` | CV improvement tool |
| Chat | `src/components/ChatPage.jsx` | Chat interface |
