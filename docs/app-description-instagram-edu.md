# Edugram - Social Learning Platform

## Overview
Edugram is a social learning platform where users can:
- Browse educational content from instructors
- Follow learning topics and practice materials
- Connect with other learners via direct messaging
- Build their learning profile and community

## Key Features

### 1. Feed
- View educational posts from instructors
- See stories and short-form content
- Like and interact with posts

### 2. Explore
- Discover different learning topics (Design, Finance, Marketing, Leadership, MBA, IELTS)
- Browse practice materials and courses
- Find new instructors

### 3. Profile
- View instructor and learner profiles
- See learning statistics and achievements
- Manage personal information

### 4. Chat
- Direct messaging with other users
- Real-time communication
- Message history

### 5. Practice
- Interactive learning carousel
- Guided practice materials
- Progress tracking

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: TailwindCSS, Radix UI
- **Backend**: Node.js Express
- **Icons**: Lucide React

## File Structure
```
src/
├── components/          # Main page components
├── components/ui/       # Reusable UI components
├── data/               # Mock data
├── services/           # API and content services
├── types/              # TypeScript definitions
├── hooks/              # Custom React hooks
└── lib/                # Utilities
```

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Development
- Development server runs on `http://localhost:5173`
- Hot Module Replacement (HMR) enabled for fast updates
- TypeScript for type safety
- ESLint for code quality
